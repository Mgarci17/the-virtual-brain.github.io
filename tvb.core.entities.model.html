tvb.core.entities.model.model_operation.Algorithm¶

An algorithm will be part of a group. The module and ClassName are those of it’s corresponding group. It will thus have a groupId, an Identifier(none if single algorithm in module or unique per algorithm is more than one algorithm launched from same group) additionally it will hold the required DataType input, the input parameter name and the outputList.
