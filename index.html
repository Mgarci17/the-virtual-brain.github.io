The Virtual Brain (TVB) is a framework for analyzing and modeling neuroimaging data.

To download it, check out the official TVB website

For installation, refer to Installation Manual

After it’s installed, see the web interface’s user guide, User Guide or, alternatively, the tutorials on using the console interface Tutorials

Lastly, when importing your own data, you may find A Description of a Complete Dataset useful.

To get in touch with other users and developers, please head over to the mailing list.
