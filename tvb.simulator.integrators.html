tvb.simulator.integrators.IntegratorStochastic¶

The IntegratorStochastic class is a base class for the stochastic integration methods. It derives from the Integrator abstract base class.

We consider a stochastic differential equation has the generic form:

X˙i(t)=dXi( X⃗)+gi( X⃗)ξ(t)

where we assume that the stochastic process ξ is a Gaussian forcing. In the deterministic case, g(X) would be zero. The full algorithm, for one external stochastic forcing which is additive, is:

Xi(t)=Xi(0)+gi(X)Z1(t)+dXit+Z2(t)dX i,kgk(X)+0.5dX i,jkgj(X)gk(X)Z3(t)+0.5t2dX i,jdXj

where Z1, Z2 and Z3 are Gaussian variables, assuming the Einstein notation and defining:

dX i,j= ∂dXi ∂Xj
