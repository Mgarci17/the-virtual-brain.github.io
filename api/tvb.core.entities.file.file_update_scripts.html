002_update_files Module¶

tvb.core.entities.file.file_update_scripts.002_update_files.upgrade(input_file)[source]¶

In order to avoid segmentation faults when updating a batch of files just start every conversion on a different Python process.

@param input_file: the file that needs to be converted to a newer file storage version.
This should be a file that still uses TVB 1.0 storage (pyTables)
