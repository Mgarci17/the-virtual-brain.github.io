class tvb.interfaces.web.entities.context_noise_configuration.ContextNoiseParameters(connectivity, default_model=None, default_integrator=None)¶

Bases: tvb.interfaces.web.entities.context_spatial.BaseSpatialContext

This class behaves like a controller. The model for this controller will be the fields defined into the init method and the view is represented by the input fields with noise values for state variables.

This class may also be used into the desktop application.
