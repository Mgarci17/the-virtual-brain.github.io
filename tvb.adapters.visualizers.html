class tvb.adapters.visualizers.phase_plane_interactive.PhasePlaneInteractive(model, integrator)¶

Bases: object

An interactive phase-plane plot generated from a TVB model object.

A TVB integrator object will be use for generating sample trajectories – not the phase-plane. This is mainly interesting for visualising the effect of noise on a trajectory.
