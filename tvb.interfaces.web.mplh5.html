class tvb.interfaces.web.mplh5.mplh5_server.ServerStarter(group=None, target=None, name=None, args=(), kwargs=None, verbose=None)¶

Bases: threading.Thread

Handler for starting in a different thread the MPLH5 server. Synchronization event. We want to start MPLH5 server in a new thread, but the main thread should wait for it, otherwise wrong import of pylb might be used.
